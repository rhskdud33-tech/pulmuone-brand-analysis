# 풀무원 냉동만두 브랜드 포지셔닝 분석

## 프로젝트 개요
풀무원 자사 브랜드의 포지셔닝이 소비자에게 
실제로 전달되는지 데이터로 검증한 프로젝트

## 데이터
- 수집처: 쿠팡, 네이버 리뷰
- 수집량: 46,020개
- 수집방법: Selenium, BeautifulSoup 크롤링

## 분석 방법
- Gemini API 감성분석
- PCA 기반 군집화
- 부정리뷰 비율 비교

## 주요 결과
- 풀무원식품만 기업의도 vs 소비자인식 불일치 확인
- "일상 속 건강함" 포지셔닝 공백 발견

## 사용 기술
Python, Pandas, Selenium, BeautifulSoup, 
Gemini API, Scikit-learn
